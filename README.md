# Pager

Simple pager to using in SwiftUI for iOS and watchOS.

## Requirements
* iOS 14.0+
* watchOS 8.0+
* Swift 5.1+

### Swift Package Manager

In Xcode:
* File ⭢ Swift Packages ⭢ Add Package Dependency...
* Use the URL https://github.com/fermoya/SwiftUIPager.git
